# Shift Management System

A simple **Shift Management System** built using **Go (Golang)** where an admin can assign shifts to users, and users can log in to view their assigned shifts in a calendar-style view.

## Features

* Admin and User login
* Admin can assign shifts
* Users can view their shifts after login
* Session-based authentication
* MVC-based project structure

## Tech Stack

* Go (Golang)
* HTML, CSS, Go Templates

## How to Run

```bash
go mod tidy
go run ./...
```

Open in browser: `http://localhost:8080`
